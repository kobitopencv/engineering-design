・OpenCVのライブラリ追加について
※環境はVisualStadioを想定しています（バージョンはなんでもいいです。多分…）
①ネットワークを繋いでください。有線無線は問いません。
②プロジェクト、.cppのソースファイルを作ります。
③上のファイル・編集・～が並んでるところのプロジェクトを開きNuGetパッケージの管理を選択します。
④右上にオンラインの検索と書かれた検索ボックスが出てくるのでそこに「OpenCV」と入力して検索します。
⑤恐らく一番上に出てくるOpenCV（2.2.0～とかバージョンが書いてないやつ）を選択しインストールボタンを押してください。
これでOpenCVのインストールは完了です
